# fangxin-image-gen-skill

Packaged Codex/Claude skill for Fangxin image generation and image editing.

Contents:

- `fangxin-image-gen/` — the skill directory
- `fangxin-image-gen.zip` — distributable zip package

Highlights:

- Text-to-image with `gpt-image-2`
- Image-to-image and multi-image merge
- URL inputs auto-downloaded to temp files, then cleaned up
- No bundled default API key

Main entry files:

- `fangxin-image-gen/SKILL.md`
- `fangxin-image-gen/scripts/generate.py`
